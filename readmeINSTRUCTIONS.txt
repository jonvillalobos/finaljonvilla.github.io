Project used React and bootstrap. Main pain is a kanto pokedex for the first 151 pokemon. Uses Poke api.
Second page is a weather app. It uses openweather api and google maps / places api.
Clicking on the pokemon brings a stats page about the pokemon. 

In the project directory, you can run:
npm start

Runs the app in the development mode.
Open http://localhost:3000 to view it in the browser.

The page will reload if you make edits.
You will also see any lint errors in the console.
